🎮 HunterGrid Game

Last Reload is a simple top-down survival shooter built with C# and Windows Forms.
The main goal is to survive as long as possible while being chased by multiple enemies.

The player can move in four directions and shoot, but there is a 10-second reload time.
This makes every shot important and forces you to think before shooting.

🕹 Gameplay Features

4-direction player movement

Enemy AI that follows the player

Non-diagonal enemy movement

Shooting system with cooldown

Enemy respawn after being eliminated

Score tracking system

Increasing difficulty over time (enemy speed scales up)

🎯 Objective

Survive, manage your reload timing, and try to reach the highest score possible.

🛠 Technical Details

Built with .NET and Windows Forms

Single-file executable build

Uses collision detection and timer-based game loop
